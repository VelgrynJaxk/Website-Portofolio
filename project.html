Project Details

Advantages of working with this IoT-based Smart Home System

This project presents an IoT-based Smart Home system built using the ESP32 microcontroller, DHT11 temperature and humidity sensor, and a website interface for real-time control and monitoring. It was implemented at Uchu Boba Café to solve real-world operational issues such as employee negligence in managing lighting and ensuring proper storage temperature for food ingredients.

  • 1
    Automatic Environmental Control

    The system automatically controls lighting and activates a fan when the temperature exceeds a certain threshold. This is achieved using the DHT11 sensor and a pre-defined threshold algorithm, ensuring the room stays at an ideal condition for food storage.

  • 2
    Remote Monitoring and Control

    Using the integrated website, the café owner or staff can control the lighting and monitor the temperature from anywhere as long as they are connected to the internet. This feature reduces manual errors and increases operational efficiency.

  • 3
    Real-Time Alerts

    The system is equipped with a buzzer that activates when the room temperature exceeds the threshold. This serves as an immediate local alert to the staff, preventing food spoilage due to heat.
