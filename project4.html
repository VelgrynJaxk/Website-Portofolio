Project Details

The Ticket

The Cinema Ticket Booking App is a mobile application designed to streamline the process of purchasing movie tickets in Indonesia. This project was developed with the goal of improving the movie-going experience by allowing users to browse films, choose showtimes, select seats, make payments, and manage their tickets all within a single application. The app offers a complete user journey — from registration to post-purchase ticket viewing — wrapped in a clean, intuitive user interface and built with usability as a top priority.

Cinema Ticket Booking App

This screen showcases the user authentication process: login, register, and profile photo upload. It includes input validation and error handling to ensure proper data entry.

Users can browse currently showing films, check detailed movie information, choose cinema locations (integrated with Google Maps), and pick dates and times for the show.

This part allows users to select seats, review a summary of their booking, choose a payment method, and complete the purchase. Upon successful transaction, a confirmation screen is shown.

Users can access their active and historical tickets. Each ticket includes showtime, seat number, date, and location, along with its status (Active/Offline).

This section allows users to manage their account details and profile photo, as well as read an "About Us" section that explains the app’s purpose. Successful edits are confirmed with a success screen.
